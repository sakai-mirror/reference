Sakai 2.2.2 Fixed Issues

The 2.2.2 release is designed to fix a single security issue, listed below, which required the removal of some test files included with the Apache Axis distribution. No Sakai code was otherwise altered for this release.

OSP Issues: http://bugs.osportfolio.org/jira
